In this Java program, I create a class named Aiport_array to store the five airports.
I modify\ied the following classes, Airplane, Airport, AirportEvent, AirportSim.
The main function is in the AirportSim class.
To run the program, try the following command in terminal
1. cd Airport
2. cd src
3. export _JAVA_OPTIONS=“-Xmx256M”
4. javac *.java
5. java AirportSim
Then there should be the result.
